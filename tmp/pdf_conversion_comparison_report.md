## 五、最终建议

### 对于 GBT 18487.1-2023 文档

**必须使用 MiniMax VLM OCR**

理由：
1. 这是纯图片 PDF，没有真实文本层
2. PyMuPDF 只能提取乱码字符
3. OpenDataloader 只能识别图片引用
4. MiniMax VLM 能准确识别中文、表格、流程图

### 对于其他文档类型

| 文档类型 | 推荐引擎 | 理由 |
|----------|----------|------|
| 纯图片 PDF（扫描件） | MiniMax VLM | 图像识别能力强 |
| 电子版 PDF（有文本层） | PyMuPDF | 快速，保留原始格式 |
| 混合 PDF（部分扫描） | 混合策略 | 先检测后分别处理 |
| 大文档 (>100 页) | 分块 + 并行 | 控制 API 调用量 |

### 推荐处理流程

```
1. 快速检测：用 PyMuPDF 检查是否有真实文本层
   ↓
2a. 有文本层且质量好 → PyMuPDF 直接提取
2b. 无文本层或乱码 → MiniMax VLM OCR
   ↓
3. 后处理：clean_doc_ir + restore_reading_order
   ↓
4. 输出：JSON + SQLite
```
