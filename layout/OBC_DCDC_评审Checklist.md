# OBC&DCDC 评审 Checklist（AI + 人工联合版）

> 适用范围：车载 OBC（PFC+LLC/CLLC）与 HV→12V/24V DCDC 的原理图与PCB评审。

## 0) 评审输入完整性（先过这个门槛）
- [ ] 原理图源文件 + PDF
- [ ] PCB源文件 + Gerber/ODB++
- [ ] Netlist + BOM + XY坐标
- [ ] 层叠（铜厚/介质厚度/阻抗目标）
- [ ] 设计规则（DRC/ERC/差分/长度匹配）
- [ ] 关键网络清单（PFC/LLC/DCDC驱动、采样、时钟、通信）
- [ ] EMC目标与标准（CISPR 25/32、车厂限值）
- [ ] 电源树与地策略说明（PGND/AGND/机壳地/PE）

---

## 1) 安规与隔离（最高优先级）
- [ ] AC侧、HV侧、LV侧分区边界清晰
- [ ] 爬电距离/电气间隙满足目标标准（含污染等级、海拔修正）
- [ ] 光耦/数字隔离器跨隔离栅布局无“跨区平行耦合”
- [ ] Y电容连接点和返回路径定义清楚（到机壳地/PE策略一致）
- [ ] 隔离槽、开窗、禁布区落实到版图而非只在图纸声明

---

## 2) 功率环路最小化（最小路径核心）
### PFC
- [ ] 整流桥—开关管—二极管—母线电容主环路面积最小
- [ ] 开关节点（SW）铜皮不过度扩展，避免大面积天线效应
- [ ] 母线薄膜电容尽量贴近主功率回路

### LLC/CLLC（OBC）
- [ ] 半桥/全桥与谐振腔回路最短、对称
- [ ] 变压器初级高dv/dt节点远离控制/采样区
- [ ] 次级整流环路紧凑，输出回流短闭环

### HV→12V DCDC
- [ ] 主开关环路、同步整流环路分别最小化
- [ ] 输出大电流路径宽且短，避免穿越信号区

---

## 3) EMC关键项（传导 + 辐射）
- [ ] 共模路径可画出来：噪声源 → 耦合 → 回路闭合点
- [ ] CM choke、X/Y电容位置与回流方向一致
- [ ] 机壳地“单点/多点”策略明确且可复现
- [ ] 高频回流参考平面连续，无跨分割缝强制绕路
- [ ] 高dv/dt节点与连接器、线束、采样线保持隔离
- [ ] 控制板与功率板间接口有滤波/地参考方案

---

## 4) 驱动与采样抗干扰
- [ ] 栅极驱动回路短且成对（GATE/RETURN）布线
- [ ] Kelvin Source / Kelvin Sense正确回归，不混入功率地电流
- [ ] 电流采样（分流器/霍尔）前端RC与地回路紧凑
- [ ] 运放/ADC参考地远离大di/dt路径
- [ ] 隔离驱动器去耦电容贴脚，回路最小

---

## 5) 去耦与电源完整性（最小距离可量化）
- [ ] 各IC电源脚去耦“就近放置”（优先看驱动、隔离器、MCU、ADC）
- [ ] 高频去耦与低频储能电容分工清晰
- [ ] 去耦回流不过过孔链绕行
- [ ] 控制电源（5V/3V3）与功率噪声有隔离手段（LDO/LC/磁珠）

---

## 6) 铜皮与过孔（最小铺铜面积相关）
- [ ] 无高频孤岛铜（floating copper）或已明确接地策略
- [ ] 大电流铜皮面积满足温升目标，且有足够并联过孔
- [ ] 热扩散路径连续，热点器件下方散热铜和导热过孔合理
- [ ] 开关节点铜面积“够用但不扩散”，避免EMI恶化
- [ ] 过孔电流能力校核（数量/孔径/铜厚）

---

## 7) 连接器与系统接口
- [ ] AC输入、HV输出、LV输出、通信口分区与走线隔离
- [ ] 线束出口附近预留EMI抑制（共模/TVS/RC）位置
- [ ] ESD/浪涌器件靠近接口第一落点
- [ ] 接口地参考路径明确，避免“保护器件接了但回路不闭合”

---

## 8) 可制造/可测试/可维护
- [ ] DFT测试点覆盖关键节点（母线、驱动、采样、通信）
- [ ] 关键器件丝印清晰，可返修空间可达
- [ ] 安规间距在Gerber层面再次自动检查
- [ ] 关键参数（阻抗、线宽线距、孔）与板厂能力匹配

---

## 9) AI输出报告格式（建议固定模板）
每个问题按以下格式输出：
1. 位置（页码/器件/网络/坐标）
2. 问题描述
3. 风险等级（高/中/低）
4. EMC/可靠性机理（为什么有风险）
5. 修改建议（可执行，给出优先顺序）
6. 复验方法（看哪项波形/预扫指标）
